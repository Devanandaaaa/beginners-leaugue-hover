Hover Buttons Demo — Color & Shape

Open this file in a browser to see interactive hover effects. Below are several examples and the CSS that drives them.

1) Color change on hover

Hover: the background smoothly swaps colors and the button lifts.

2) Shape morph

Hover: border-radius animates to create a pill shape.

3) Icon rotate + color swap

Hover: icon rotates while the button background switches to a gradient.

4) Outline → Animated fill

Hover: a pseudo-element slides in to fill the button with color.

5) 3D lift & shadow

Hover: subtle 3D transform and deeper shadow for a tactile feel.
